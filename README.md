# BEETREE _*beta*
> A very easy and fast web client for ***MySQL Database Management***


## Requirements

* **PHP** *v5.5.0+*
* **MySQL** *v5.6.0+*

## Installation
Copy it to your web folder and run. For xampp copy it to htdocs folder, for wampp to www folder.
